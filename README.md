
# Capstone Project
AI multi-agent orchestration backend built using Node.js, Express, and RabbitMQ.

## Features
- REST API orchestration
- Independent AI agents
- RabbitMQ communication
- Async task handling
- Status endpoint

## Technologies
- Node.js
- Express.js
- RabbitMQ
- Postman

## Architecture
GUI → API → RabbitMQ → Agents → Results Queue → Status Endpoint
